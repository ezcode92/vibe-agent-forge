# RDB

## 목적

관계 모델, transaction, constraint, index, migration과 query 검증 원칙을 담당합니다.

## 포함된 문서 자산

- RDB 전용 `agents.fragment.md`
- framework 연동 bridge에 필요한 관계형 데이터 기준

## 현재 범위

`agents.fragment.md`는 RDB 지침 fragment로 존재하며 `registry/fragments.yml`에서 조회할 수 있습니다. Schema, migration과 특정 DB 제품 설정은 포함하지 않습니다.
