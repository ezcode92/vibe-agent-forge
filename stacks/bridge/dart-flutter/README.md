# Dart–Flutter Bridge

## 목적

Dart의 null safety와 비동기 model을 Flutter widget state 및 lifecycle에 연결하는 조합 책임을 안내합니다.

## 포함된 문서 자산

- `agents.fragment.md`: Dart–Flutter bridge fragment
- Catalog 항목: `registry/fragments.yml`의 `bridge-dart-flutter`

## 관련 Profile과 Dry-run

- `profiles/flutter-app/profile.yml`
- `examples/codex/flutter-app/`

## 현재 범위

이 디렉터리는 두 stack 사이의 조합 규칙을 제공하는 문서 자산입니다. Flutter application, package 설정과 platform 코드는 포함하지 않습니다.
