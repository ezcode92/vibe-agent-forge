# JavaScript–React Bridge

## 목적

JavaScript object·async 규칙을 React props·state·effect와 rendering lifecycle에 연결하는 조합 책임을 안내합니다.

## 포함된 문서 자산

- `agents.fragment.md`: JavaScript–React bridge fragment
- Catalog 항목: `registry/fragments.yml`의 `bridge-javascript-react`

## 관련 Profile과 Dry-run

- `profiles/frontend-react/profile.yml`, `profiles/fullstack-spring-react/profile.yml`
- `examples/codex/frontend-react/`, `examples/codex/fullstack-spring-react/`

## 현재 범위

이 디렉터리는 두 stack 사이의 조합 규칙을 제공하는 문서 자산입니다. React component, package 설정과 client 코드는 포함하지 않습니다.
