# Flutter–REST API Bridge

## 목적

REST API와 Flutter client 사이의 데이터 변환, error 처리와 호환성 계약을 연결합니다.

## 포함된 문서 자산

- Flutter–REST API 전용 bridge `agents.fragment.md`
- mobile client와 API의 계약 검증 기준

## 현재 범위

`agents.fragment.md`는 Flutter–REST API bridge fragment로 존재하며 `registry/fragments.yml`에서 조회할 수 있습니다. API schema와 Flutter client 코드는 포함하지 않습니다.
