# Spring–RDB Bridge

## 목적

Spring transaction·repository 경계와 RDB constraint·migration·query 원칙을 연결합니다.

## 포함된 문서 자산

- Spring–RDB 전용 bridge `agents.fragment.md`
- transaction과 영속성 검증 기준

## 현재 범위

`agents.fragment.md`는 Spring–RDB bridge fragment로 존재하며 `registry/fragments.yml`에서 조회할 수 있습니다. ORM 설정, schema와 migration은 포함하지 않습니다.
