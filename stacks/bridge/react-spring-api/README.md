# React–Spring API Bridge

## 목적

Spring REST API와 React client 사이의 request·response·error 및 호환성 계약을 연결합니다.

## 포함된 문서 자산

- React–Spring API 전용 bridge `agents.fragment.md`
- backend와 frontend의 계약 검증 기준

## 현재 범위

`agents.fragment.md`는 React–Spring API bridge fragment로 존재하며 `registry/fragments.yml`에서 조회할 수 있습니다. API schema, endpoint와 client 코드는 포함하지 않습니다.
