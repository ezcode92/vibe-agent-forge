# Kotlin–Spring Bridge

## 목적

Kotlin 관용구와 Spring의 bean·proxy·transaction 모델 사이의 결합 규칙을 담당합니다.

## 포함된 문서 자산

- Kotlin–Spring 전용 bridge `agents.fragment.md`
- 적용에 필요한 Kotlin과 Spring 참조 정보

## 현재 범위

`agents.fragment.md`는 Kotlin–Spring bridge fragment로 존재하며 `registry/fragments.yml`에서 조회할 수 있습니다. Spring 설정과 Kotlin 코드는 포함하지 않습니다.
