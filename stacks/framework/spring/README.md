# Spring

## 목적

Spring의 bean 책임, 계층 경계, 설정, transaction, validation과 test 관례를 담당합니다.

## 포함된 문서 자산

- Spring 전용 `agents.fragment.md`
- Java·Kotlin 및 RDB 조합에 필요한 bridge 참조 정보

## 현재 범위

`agents.fragment.md`는 Spring 지침 fragment로 존재하며 `registry/fragments.yml`에서 조회할 수 있습니다. 애플리케이션 설정과 예제 코드는 포함하지 않습니다.
