# Hexagonal Architecture

## 목적

port와 adapter 경계를 통해 domain의 외부 의존성을 차단하는 원칙을 담당합니다.

## 포함된 문서 자산

- Hexagonal Architecture 전용 `agents.fragment.md`
- port·adapter 책임과 의존성 검증 기준

## 현재 범위

`agents.fragment.md`는 Hexagonal Architecture 지침 fragment로 존재하며 `registry/fragments.yml`에서 조회할 수 있습니다. Package template과 adapter 구현은 포함하지 않습니다.
