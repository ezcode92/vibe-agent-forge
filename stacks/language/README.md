# language

## 목적

프레임워크와 무관한 언어별 관례, 타입·모듈 구조와 기본 도구의 책임 범위를 관리합니다.

## 포함된 문서 자산

- Java, Kotlin, Python, JavaScript, Dart별 `agents.fragment.md`
- 언어 고유 지침의 메타데이터와 검증 기준

## 현재 범위

Java, Kotlin, Python, JavaScript와 Dart fragment가 존재하며 `registry/fragments.yml`에서 조회할 수 있습니다. 언어별 빌드·package 구현은 포함하지 않습니다.
