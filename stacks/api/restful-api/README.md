# RESTful API

## 목적

resource 중심 URI, HTTP method·status, request·response·error 계약과 호환성 원칙을 담당합니다.

## 포함된 문서 자산

- RESTful API 전용 `agents.fragment.md`
- server framework와 client를 연결할 bridge 기준

## 현재 범위

`agents.fragment.md`는 RESTful API 지침 fragment로 존재하며 `registry/fragments.yml`에서 조회할 수 있습니다. OpenAPI 문서, endpoint와 client 코드는 포함하지 않습니다.
