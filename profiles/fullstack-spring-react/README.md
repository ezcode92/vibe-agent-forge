# fullstack-spring-react

## 목적

Spring backend와 React frontend를 RESTful API 계약으로 연결하는 fullstack 구성 조합을 설명합니다.

## 포함 예정 파일

- backend·frontend stack과 React–Spring API bridge 참조를 담는 profile manifest
- backend language variant와 호환성 제약 메타데이터

## 현재 범위

Phase 1에서는 조합 책임만 정의하며 실제 YAML, API와 애플리케이션 코드는 작성하지 않습니다.
